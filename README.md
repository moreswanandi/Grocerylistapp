Add items, mark them as done, and delete them when no longer needed — all in one clean interface.

## 📘 About the Project
The Grocery List App is designed to make shopping stress-free.  
No more forgetting items or juggling paper notes — everything is digital, interactive, and easy to use.

---

## ⚙️ How It Works
- ✍️ **Enter Item**: Type groceries into the input box.  
- ➕ **Add Item**: Click *Add* to place it into the list.  
- ✅ **Mark Done**: Check off items once purchased.  
- 🗑️ **Delete Item**: Remove items you don’t need anymore.  
- 🔄 **Update List**: The list refreshes automatically.  
- 📋 **Save & Review**: Keep track of your shopping history.
